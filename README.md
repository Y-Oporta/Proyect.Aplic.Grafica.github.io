# Proyecto Restaurante 🍽️

## Descripción del Proyecto

Este proyecto consiste en el desarrollo de una página web para un restaurante. Hemos elegido esta temática porque nos permite aplicar de manera práctica los conocimientos y habilidades adquiridos en clase, integrando diferentes herramientas y tecnologías aprendidas durante el curso.

Además, este tipo de sitio web facilita la incorporación de diversos recursos multimedia, como imágenes de los platillos, diseño visual atractivo, estilos personalizados y funcionalidades interactivas con JavaScript. De esta manera, cumplimos con los objetivos académicos y demostramos lo aprendido.

La idea surge también a partir de la visita a un restaurante que nos dejó una experiencia muy positiva, tanto por la excelente atención como por la calidad de la comida. Tomamos esa experiencia como inspiración para desarrollar nuestra propia propuesta web.
